# HexNowYouRest

Get the Rested buff in 2 seconds instead of 20.

## Features

- Get the Rested buff in 2 seconds instead of 20.
- Receive the Rested buff sooner
- No configuration required

---

## Installation

### Thunderstore

Install with your preferred mod manager.

### Manual Installation

1. Install BepInEx for Valheim.
2. Extract `HexNowYouRest.dll` into:

```text
BepInEx/plugins/
```

3. Launch the game.

---

## Multiplayer

This mod has not been tested in multiplayer.

Because the Rested buff is applied locally, all players who want the reduced Resting delay should install the mod.

---

## Compatibility

Any mod that modifies the `SE_Cozy` Resting delay may conflict.
